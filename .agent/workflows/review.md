# /review - QA 검토

## 검토 유형

### 코드 리뷰

```
트리거: "/review", "코드 리뷰해줘"

검토 항목:
1. 코드 품질
   - 명명 규칙
   - 함수 길이
   - 중복 코드

2. Rust 규칙
   - unsafe 사용 여부
   - 에러 처리
   - 라이프타임

3. 테스트
   - 테스트 존재 여부
   - 커버리지

4. 문서화
   - rustdoc 주석
   - README 업데이트 필요 여부
```

### 보안 검사

```
트리거: "보안 검사", "취약점 확인"

검사 항목:
1. 의존성 취약점
   - cargo audit

2. 코드 취약점
   - SQL 인젝션
   - XSS
   - 하드코딩된 시크릿

3. 설정 취약점
   - 약한 암호화
   - 안전하지 않은 기본값
```

### 성능 검토

```
트리거: "성능 검토", "최적화 제안"

검토 항목:
1. 알고리즘 복잡도
2. 불필요한 할당
3. 동시성 이슈
4. 데이터베이스 쿼리
```
