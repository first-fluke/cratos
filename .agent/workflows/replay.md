# /replay - 되감기

## 3가지 모드

### ViewOnly (기본)

```
트리거: "방금 한 거 보여줘", "실행 기록"

프로세스:
1. 실행 ID 확인 (최근 또는 지정)
2. EventStore에서 이벤트 로드
3. 타임라인 형태로 렌더링
```

출력:
```
┌─────────────────────────────────────────────────────────────┐
│ 🔄 리플레이: Issue #123 수정 작업                           │
├─────────────────────────────────────────────────────────────┤
│ [00:00] 📥 입력 수신                                        │
│         "이 GitHub 이슈 고쳐서 PR 만들어줘"                 │
│ [00:02] 📝 계획 수립                                        │
│ [00:05] 🔧 도구 호출: git_clone ✅                          │
│ [00:08] 🔧 도구 호출: file_read ✅                          │
│ [00:12] 🔧 도구 호출: file_write ✅                         │
│ [00:15] 🔧 도구 호출: exec (cargo test) ✅                  │
│ [00:20] 🔧 도구 호출: github_pr ✅                          │
│ [00:22] 📤 최종 응답                                        │
└─────────────────────────────────────────────────────────────┘
```

### Rerun

```
트리거: "다시 실행해줘", "같은 거 다시"

프로세스:
1. 기존 실행의 입력 추출
2. 새 실행 ID 생성
3. 동일 조건으로 재실행
4. 결과 비교 (선택)
```

### DryRun

```
트리거: "dry-run으로 보여줘", "실행 안 하고 계획만"

프로세스:
1. 기존 입력 추출
2. 계획 단계까지만 실행
3. 실제 변경 없이 "무엇을 할지" 출력
```

출력:
```
┌─────────────────────────────────────────────────────────────┐
│ 🔄 Dry-Run: Issue #123 수정                                 │
├─────────────────────────────────────────────────────────────┤
│ ⚠️ 시뮬레이션입니다. 실제 변경 없음.                       │
├─────────────────────────────────────────────────────────────┤
│ 계획된 단계:                                                │
│ 1. [Read] git_clone: user/repo                             │
│ 2. [Read] file_read: src/handler.rs                        │
│ 3. [Write] file_write: src/handler.rs (+5 -2)              │
│ 4. [Read] exec: cargo test                                 │
│ 5. [Write] github_pr: Fix #123                             │
├─────────────────────────────────────────────────────────────┤
│ [▶️ 실행] [❌ 취소]                                         │
└─────────────────────────────────────────────────────────────┘
```

## 실행 목록 조회

```
트리거: "최근 실행 목록"

출력:
┌────────────────────────────────────────────────────────────┐
│ 📋 최근 실행 목록                                          │
├────────────────────────────────────────────────────────────┤
│ ID       │ 시간            │ 요약              │ 상태      │
├──────────┼─────────────────┼───────────────────┼───────────┤
│ abc123   │ 10분 전         │ Issue #123 수정   │ ✅ 성공   │
│ def456   │ 1시간 전        │ 파일 정리         │ ✅ 성공   │
│ ghi789   │ 2시간 전        │ S3 업로드         │ ❌ 실패   │
└────────────────────────────────────────────────────────────┘

"abc123 보여줘" 또는 "ghi789 왜 실패했어?" 로 상세 조회
```
